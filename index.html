Aplikacija je odprta direktno iz datoteke (file://) — API klici ne bodo delovali!
Brskalnik iz varnostnih razlogov blokira povezave na internet, ko je stran odprta na ta način (vidi se "Failed to fetch").

Rešitev — uporabi eno od teh možnosti:
  1. Najlažje: v brskalniku Chrome pojdi na chrome://extensions, vklopi "Razvijalski način" ni potrebno — namesto tega povleci to HTML datoteko v nov zavihek in preveri ali naslov začne z file://. Če da, glej spodnjo možnost.
  2. Priporočeno: prosi Borisa, naj ti pošlje povezavo do aplikacije gostovane na spletu (npr. GitHub Pages ali podoben brezplačen gostujoč strežnik) namesto same datoteke.

📋 Kako pridem do API ključa?
  1. Odpri console.anthropic.com.
  2. Če nimaš računa: klikni Sign Up, vnesi e-mail in geslo, potrdi e-mail.
  3. V levem meniju klikni API Keys → Create Key.
  4. Vnesi ime (npr. MSORA Obračun), potrdi. Ključ se prikaže samo enkrat — ga takoj skopiraj.
  5. Prilepi ključ v polje zgoraj in klikni Shrani.
  6. V levem meniju klikni Billing → Credits kaufen → dodaj vsaj 10 $ (~250 obračunov).
⚠️ Ključ se shrani samo lokalno v tem brskalniku. En skupen ključ za vse uporabnike M SORA je v redu (~0,04 EUR/obračun).
